# Garden Lamp

Controller: Adafruit Feather HUZZAH ESP8266 + `FauxMoESP`

This lamp is modified from a store-bought garden lamp that contains:
- solar panel array
- rechargeable NiMH battery pack (3 AA)
- light sensor (possibly a photodiode)
- LED
